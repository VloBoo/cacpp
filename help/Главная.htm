На тему "Анализатор кода С++"

Данная справка предназначена для курсового проекта на тему "Aнализатора кода С++". Справка содержит из себя 3 страницы. Каждая последующая страница содержит некоторую информацию о решение основных и наиболее часто встречаемых проблем или вопросов.

Локализация

Программа предусматривает три языка локализации: английский, русский и беларуский. OS Windows автоматически выставляет необходимую локализацию в соотвествии с локализацией операционной системы. Но при этом пользователь может изменить локализацию, которая будет распостроняться на всю программу.

Справка

В любом состоянии пользователь может вызвать данную справку нажатием клавиши клавиатуры "F1".
