Анализатор кода

Работа с файлом

Оснавная работа с анализатором сводится к тому, что пользователь должен выбрать или создать файл, где в последующем будет находится обрабатываемый код.

Пользователь может выбрать файл в контекстном меню (Файл>Открыть файл...) открыть файл или же создать новый файл в контекстном меню (Файл>Новый файл...). По умолчанию, при открыти приложения, создается новый файл. Пользователь может изменять код в окне файла и сохранять его посредством использования контекстнога меню (Файл>Сохранить файл...) или (Файл>Сохранить файл как...).

Работа с анализатором кода файла

Пользователь может выбрать в контекстном меню (Файл>Анализ), который запустит алгоритм анализа кода файла. В конце работы алгоритма будет выведина информация о результатах алгоритма в правом окне, где будет выведина информация в соответствии с названиями закладок. Закладка "Функции" будет отсортированна по имени. Так же вы можете выбрать функции и по щелкчу выше курсор онка файла будет установлен в начало имени функции.

Так же в конце работы анализатор подсвечивает слова в соответствии с результатами анализа. Синий, красный, фиолетовый для ключевых слов, функций, переменных соответстенно. При изменении текста в окне файла или другого заданного действия, цвета результата работа анализатора будут сброшены, так как не являются актуальными, а закладки левого окна заблокированны.

Выход из программы

Программа предусматривает выход, для корректного завершения программы, в контекстном меню (Файл>Выход). Кроме того пользователь может выйти в меню, регистрации в контекстном меню (Файл>Выйти в меню регистрации), где последующем сможет завоно зайти в программу.

Шрифт

Программа предусматривает изменения шрифта окна файла в контекстном меню (Вид>Шрифт...). Стоит заметить, что изменение шрифт сбрасывает цвет результата работы анализатора.

Фон

Программа предусматривает изменения фона в контекстном меню (Вид>Фон...). На выбор предоставляется 3 цвета: белый, голубой, розовый. По умолчанию установлен белый фон программы.

Панель состояния

Панель состояния находится в нижней части программы и содержит информацию состояния. В панели состояния указан активный язык приложения, через черту указано положение курсора в окне файла.
